Create your first React App!

In this exercise you are gonna create, by using components, your own book store!
Install and use the create-react-app command to create your new application
Install and configure Boostrap / react-bootstrap in your app
Create a MyNav component which include a boostrap navigation bar (Home, About, Browse) Note: we don't know how to navigate, so please just put some "#" in the hrefs
Create a MyFooter component which include a footer for the page
Create a Welcome component with a Jumbotron and a tagline for your shop
Create a Latest Release component. This component should read from one of the json files and print out the cover for the books
Publish everything on GitHub
